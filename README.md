# HitArrowRangeRemake
記録用のAddon

# 基本的な使い方
#### ベータAPIを有効にしてください<br>
　ScoreBoard「HitDistance」にPLayerの記録と最高、最低記録が記録されます。<br>
 誰かにTag「reset」を付けると記録がリセットされます。
 
# ダウンロード

PC　右のReleaseから<br>

スマホ　下or上のReleaseから<br>

下の〇〇.mcaddon(.mcpack)っていう青い奴をクリックすると、 ダウンロードできます！<br>
